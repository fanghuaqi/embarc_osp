Detailed Description

Test FreeRTOS working with secureshield runtime.

Extra Required Peripherals

Design Concept

This example is designed to show how FreeRTOS is integrated with SecureShield. Here, trap_s is used to trigger a task context switch request. So application should not use trap exception.

As container switch is not with task switch. So during a container call, task scheduler should be suspended to avoid a task context.

Usage Manual

Here we take EMSK 2.3 EM7D for example, you can run the program using Metaware toolset.

  • Re-configure the EMSK 2.3 board to EM7D configuration.
  • Command: gmake BOARD=emsk BD_VER=23 CUR_CORE=arcem7d run
