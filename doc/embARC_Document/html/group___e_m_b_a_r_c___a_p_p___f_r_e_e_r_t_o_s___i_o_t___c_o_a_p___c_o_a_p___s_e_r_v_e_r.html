Detailed Description

embARC Example for CoAP Protocol on FreeRTOS

Extra Required Tools

  • Firefox with CoAP Plugin

Extra Required Peripherals

  • Digilent PMOD WIFI(MRF24WG0MA)
  • Digilent PMOD TMP2

Design Concept

This example is designed to show how CoAP server running on FreeRTOS.

Usage Manual

This is an example running coap server on LwIP with FreeRTOS. The Pmod modules should be connected to EMSK.

  • This is a example similiar but simpler than the contiki coap_server one, so you can almost following the instruction list in contiki coap_server example, you need to check if you can ping emsk board, sometimes you need to clear arp table by run arp -d.
  • Open Firefox and enter coap://<your board="" ip="" address>="">, and then you can test the coap server example.
  • Know issues: temperature obeserve will hang up.
