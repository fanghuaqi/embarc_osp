Detailed Description

AXS Board Common Init Module.

AXS timer/gpio/interrupt init process. Device-driver installation is done while getting the device object for the first time.

Function Documentation

void board_delay_ms ( uint32_t  ms,
uint8_t  os_compat 
)

provide MS delay function

this function needs a 1-MS timer interrupt to work. For bare-metal, it is implemented in this file. For OS, you must call board_timer_update in the OS 1-MS timer interrupt when the bare-metal timer interrupt is not ready

Parameters
[in]msMS to delay
[in]os_compatEnable or disable When this delay is enabled, use the OS delay function, if one is provided. See OSP_DELAY_OS_COMPAT_ENABLE and OSP_DELAY_OS_COMPAT_DISABLE
