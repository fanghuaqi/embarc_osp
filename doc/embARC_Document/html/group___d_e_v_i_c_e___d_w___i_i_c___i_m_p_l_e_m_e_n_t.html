uint32_t  mode,
uint32_t  param 
)

open a designware iic device

Parameters
[in]iic_objiic device object pointer
[in]modeiic working mode (master or slave)
[in]paramWhen mode is DEV_MASTER_MODE, param stands for speed mode, when mode is DEV_SLAVE_MODE, param stands for slave device 7bit address
Return values
E_OKOpen successfully without any issues
E_OPNEDIf device was opened before with different parameters, then just increase the opn_cnt and return E_OPNED
E_OBJDevice object is not valid
E_SYSDevice is opened for different mode before, if you want to open it with different mode, you need to fully close it first.
E_PARParameter is not valid
E_NOSPTOpen settings are not supported
